# 2026 AI Science Fellowship — Fellow Projects

A running archive of the projects fellows build each week. Every `Week N/` folder is a
**point-in-time snapshot** of each fellow's project repository at the end of that week — the
repos are vendored as plain files (their `.git` history is stripped), so they are frozen
copies, not live submodules. For the latest version of any project, follow the fellow's GitHub
link. The same project can appear in multiple weeks, showing how it evolved over time.
